#### Консольный набор мини-игр, запускаемых из консоли. Эти игры похожи на то, что обычно называют "Brain Games".  

*Установка приложения:*  
`npm install -g brain-games-by-yarik`  

*Использование:*  
`$ brain-even`  
`$ brain-calc`  
`$ brain-gcd`  
`$ brain-balance`  
`$ brain-prime`  
`$ brain-progression`
